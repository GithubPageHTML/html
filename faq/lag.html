Fix Android laggy by another option

There are some steps that help speed up the Android system to its fullest

Battery Settings

please go to the battery settings. There is a sleep mode option. My Check all the apps you don't use

Data Saver Mode

there is a data saver mode. Deselect all applications except the important ones

Bug no response

Ensure that the word error does not appear in the application. Ensure that the following options are enabled in the developer options. To ensure that all installed applications do not contain any bug
- Always down notifications
- View the media channel
- Show invalid channel notic

Photo Enhancement

disable Force 4x MSAA from developer option:
disable HW overlays because it's slow dawn your phone
Without a hardware overlay every application that is displaying things on the screen will share video memory and will have to constantly check for collision and clipping to render a proper image, this can cost a lot of processing power. With a hardware overlay each application gets its own portion of video memory, getting rid of the need to check for collision and clipping

Process in background

From the developer options, set the number of background processes on 4

Running Services

There are also two sections in the developer options The first is:
Running services:
which are permanent servers that consume the CPU
Background check:
which is the apps that run in RAM

Android Hidden Codes

If you have a phone that has this option Dial the code *#9900# and select the option shown in the image, then choose low not high It configures how many logs the phone stores for debug reasons if you choice high Your phone starts writing more logs than usual

Internet

Download adguard and block some apps from internet to completely turn the internet source into VMOS App

Stop Force

use the Greenify program or Force Stop apps to disable applications before entering the game

Disable app

It is preferable to disable Google applications if you do not use them and if your phone system interface allows you to do so
